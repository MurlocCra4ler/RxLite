Detailed Description

template<typename T>
class RxLite::Observer< T >

A concrete Observer class for a specific type.

The Observer class represents an observer that handles a specific type of data (e.g., int, float). Observers are simply a set of callbacks, one for each type of notification delivered by the Observable:

  • next for handling emitted values
  • error for handling errors
  • complete for handling completion signals.

The observer pattern is used to allow an object to react to changes in another object (the observable) without tightly coupling the two.
