Detailed Description

template<typename T>
class RxLite::Subscriber< T >

A Subscriber that observes events from an Observable.

A Subscriber acts as an observer that receives values, errors, and completion signals from an Observable. Once an error or completion signal is received, the subscriber is marked as inactive and will not process further events.

Template Parameters
TThe type of data received by the subscriber.
