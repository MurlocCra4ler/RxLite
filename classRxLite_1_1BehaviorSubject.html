Detailed Description

template<typename T>
class RxLite::BehaviorSubject< T >

A special type of Subject that replays the last emitted value to new subscribers.

BehaviorSubject<T> is a variant of Subject<T> that stores the most recent value and emits it immediately to any new subscriber upon subscription. This ensures that late subscribers always receive the latest known value.

Template Parameters
TThe type of values emitted by this subject.

Emits an error to all subscribers.

This method notifies all subscribers that an error has occurred, thereby terminating the observable sequence. After calling error(), any subsequent calls to next(), complete(), or further invocations of error() will have no effect.
