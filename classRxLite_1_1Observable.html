Detailed Description

template<typename T>
class RxLite::Observable< T >

Represents a sequence of values over time.

An Observable<T> is the core building block of RxLite. It defines a data stream that can emit values of type T, an error, or a completion signal. Observables can be subscribed to, transformed, and composed using operators.

Template Parameters
TThe type of values emitted by this observable.
