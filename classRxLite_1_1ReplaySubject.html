Detailed Description

template<typename T>
class RxLite::ReplaySubject< T >

A variant of Subject that replays previously emitted values to new subscribers.

ReplaySubject stores the last bufferSize values and emits them to any new subscriber before subscribing it to future values. This allows late subscribers to receive past values.

Template Parameters
TThe type of values emitted by the ReplaySubject.
